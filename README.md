# ✨ Novel Forge - 你的故事冲突灵感锻造炉 ✨
还在对着空白文档抓头发？想不出有张力的故事冲突？灵感卡壳到想摆烂？

别慌！Novel Forge 是一款**纯前端、离线运行**的故事冲突灵感工具——没有套路满满的AI代写，只有「随机但讲逻辑」的叙事链生成，帮你把零散的脑洞，锻造成有血有肉的故事冲突～

## 🔥 核心亮点：不止是“随机”，更是“有灵魂”
- 📜 「结构化脑洞」：输入关键词/主角身份/题材，一键生成「源头→欲望→阻碍→行动→后果→代价→揭示→反转→结局」的完整叙事链，逻辑闭环不跑偏；
- 🌿 「多分支脑洞」：同一个故事起点，能解锁N种不同选择的结局分支，再也不愁“故事只有一条路”；
- 📦 「灵感回收站」：点击生成的灵感卡片，直接回填到输入区复用；输入的关键词会自动存进本地「灵感池」，首页还会弹出“灵感气泡”（最多飘8个新脑洞，藏18个过往灵感），不怕脑洞飞了；
- 🔒 「隐私安全感拉满」：所有草稿、历史记录、灵感数据只躺在你浏览器的本地存储里，无后端、不上传、不偷窥，断网也能玩，社恐创作者狂喜～

## 🛠️ 怎么玩？3步解锁你的专属故事
1. **填脑洞种子**：输入3-5个有张力的关键词（比如“海边小镇+失忆医生+旧灯塔+暴雨”，别只输单个词！），选好题材、冲突强度；
2. **一键锻造**：点击生成按钮，坐等叙事链出炉，还能切换分支看不同结局；
3. **盘活灵感**：喜欢的脑洞卡片点一点，回填到输入区继续打磨；灵感够多了就复制/导出文本，直接塞进你的创作文档～

## 💡 创作小Tips：脑洞更炸的秘诀
- ✨ 关键词要“带情绪”：别输“猫”，输“雨夜迷路的橘猫”；别输“小镇”，输“藏着秘密的海边小镇”，生成的冲突更有画面感；
- ✨ 冲突强度拉满：写虐文/悬疑的宝子，把冲突强度拉到最高，结局反转能让你直呼“绝了”；
- ✨ 离线也能造：把文件存本地，断网、没网、咖啡馆没信号？照样锻造脑洞，不耽误创作KPI～

## 🎨 小提醒：友好使用不踩坑
- 🚫 商用替换插画：如果要把工具公开/商用，记得把 `Isabelle_AF.png` 换成原创图或有授权的素材；
- 🗑️ 数据仅本地：清空浏览器站点数据会删掉你的灵感池，重要脑洞记得导出备份；
- 🎉 无环境依赖：不用装Node/Python/任何插件，打开浏览器就能玩，新手也能秒上手～
