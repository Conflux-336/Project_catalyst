# ✨ Novel Forge - 你的故事冲突灵感锻造炉 ✨

还在对着空白文档抓头发？想不出有张力的故事冲突？灵感卡壳到想摆烂？

别慌！Novel Forge 是一款**纯前端、离线运行**的故事冲突灵感工具——没有套路满满的 AI 代写，只有「随机但讲逻辑」的叙事链生成，帮你把零散的脑洞，锻造成有血有肉的故事冲突～

## 🔥 核心亮点：不止是“随机”，更是“有灵魂”

- 📜 **「结构化脑洞」**：输入关键词 / 主角身份 / 题材，一键生成「源头 → 欲望 → 阻碍 → 行动 → 后果 → 代价 → 揭示 → 反转 → 结局」的完整叙事链，逻辑闭环不跑偏；
- 🌿 **「多分支脑洞」**：同一个故事起点，能解锁 N 种不同选择的结局分支，再也不愁“故事只有一条路”；
- 🧭 **「导图式时间线」**：分支不再只是文本列表，可以切到导图模式，拖动画布、拖动节点、展开 / 折叠分支，一眼看清剧情走向；
- ⚖️ **「结局分支对比」**：选择两个不同结局并排查看，自动高亮“选择分歧点”和“关键情节差异”，适合判断哪条线更有戏；
- ✍️ **「批注与二次打磨」**：可以给某个分支节点写批注，或者直接修改节点内容；不满意还可以从当前节点重新生成后续分支；
- 📦 **「灵感回收站」**：点击生成的灵感卡片，直接回填到输入区复用；输入的关键词会自动存进本地「灵感池」，首页还会弹出“灵感气泡”（最多飘 8 个新脑洞，藏 18 个过往灵感），不怕脑洞飞了；
- 🔒 **「隐私安全感拉满」**：所有草稿、历史记录、灵感数据只躺在你浏览器的本地存储里，无后端、不上传、不偷窥，断网也能玩，社恐创作者狂喜～

## 🛠️ 怎么玩？3 步解锁你的专属故事

1. **填脑洞种子**：输入 3-5 个有张力的关键词（比如“海边小镇 + 失忆医生 + 旧灯塔 + 暴雨”，别只输单个词！），选好题材、冲突强度；
2. **一键锻造**：点击生成按钮，坐等叙事链出炉，还能切换分支看不同结局；
3. **盘活灵感**：喜欢的脑洞卡片点一点，回填到输入区继续打磨；灵感够多了就复制 / 导出文本，直接塞进你的创作文档～

## 🧩 分支时间线怎么玩？

- **纵向模式**：适合顺着读，像看完整剧情大纲；
- **横向模式**：适合扫视多条选择线，观察故事从哪里开始分叉；
- **导图模式**：适合编辑结构，可以拖动画布、拖动节点、折叠节点、添加批注；
- **对比模式**：选择两个结局线并排比较，看看它们是在哪里走岔的，后续又差在哪；
- **重生成后续**：在某个节点上点“重生成后续”，可以保留前面设定，只替换它后面的剧情走向。

小提示：如果你已经在某个节点写了“这里可以埋伏笔”，再重生成后续，新的分支会更适合作为后续打磨对象。

## 💡 创作小 Tips：脑洞更炸的秘诀

- ✨ **关键词要“带情绪”**：别输“猫”，输“雨夜迷路的橘猫”；别输“小镇”，输“藏着秘密的海边小镇”，生成的冲突更有画面感；
- ✨ **冲突强度拉满**：写虐文 / 悬疑的宝子，把冲突强度拉到最高，结局反转能让你直呼“绝了”；
- ✨ **导图别只看，拖起来**：把你最喜欢的分支节点拖到一边，给它加批注，再和另一个结局对比，通常很快能看出哪条线更值得扩写；
- ✨ **离线也能造**：把文件存本地，断网、没网、咖啡馆没信号？照样锻造脑洞，不耽误创作 KPI～

## 🎨 小提醒：友好使用不踩坑

- 🚫 **商用替换插画**：如果要把工具公开 / 商用，记得把 `Isabelle_AF.png` 换成原创图或有授权的素材；
- 🗑️ **数据仅本地**：清空浏览器站点数据会删掉你的灵感池、批注和历史记录，重要脑洞记得导出备份；
- 🎉 **无环境依赖**：不用装 Node / Python / 任何插件，打开浏览器就能玩，新手也能秒上手～
